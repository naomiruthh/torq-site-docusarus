Use the Data operator to efficiently filter and transform large volumes of data. The operator offers a transformation wizard which includes some of the most useful data manipulation options and a visual representation of the transformation.

Note
At this stage, the transformation wizard can only be used to manipulate data in array format. Additional functionality is coming up.

How to use

  1. Drag the Data operator onto the designer.
  2. Select Transform Data to open the transformation wizard.
  3. Select the step with the data you want to manipulate.
  4. Select the array you want to manipulate. Click Next.

Filters

Filter the data to keep only the relevant entries by applying conditions to any of the array entry values.

Apply OR/AND logic to the conditions to create a more complex filter.

Collapse the Preview array to see how many entries you're left with after applying the filter to the Original array.

Transformations

Change the structure of the array entries according to your needs.

  • Include only the keys that contain relevant data.
  • Rename keys for clarity.
  • Apply functions to the values: text can be turned to all lowercase or all uppercase and data can be split by a delimiter to keep only the relevant part.
