In some cases, you might need to import a step as a YAML file. For example, if you want to edit the step's YAML file directly or if a step is created in one account and you want to use it in a different account. 

In order to import a step to a different account, you have to export the workflow where the step is used and then import the workflow to the other account. 

1. From the account where you want to export the workflow, locate the workflow and click the three-dot menu.

2. Select Export.

3. Save the generated YAML file.

4. Go to the account where you want to import the steps and go to the Workflows page. 

5. Click the Import button and select the YAML file you previously saved. The workflow will open in the designer. 

6. Select the step you want to use in this account and in the configuration window click the three dots and select Save as a custom step.
