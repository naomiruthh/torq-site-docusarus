Use a schedule trigger when you want a workflow to run at a specific date and/or time.

Some common use cases for a schedule trigger are:

  • Onboarding/offboarding users in your organization on a regular basis, for example, once a week.
  • Querying a threat intel feed at the same time every day.

How to use

  1. Select the schedule trigger for the workflow.
    1. New workflow: after clicking Create Workflow, select the schedule trigger.
    2. Existing workflow: click the replace button above the trigger and select the schedule trigger.
  2. Set the trigger frequency. Sub-options will change according to your selection.
  3. Set the trigger time.
  4. The trigger event will include the timestamp.
