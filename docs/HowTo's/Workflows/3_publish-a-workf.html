Publish a {{glossary.Workflow}} when it's ready for production. Do a few Test runs before you publish a workflow to verify that it's running as expected. 

  1. Go to the Workflows page.
  2. Open the workflow you want to publish.
  3. Click Publish.
    1. Add a description explaining this workflow version.
    2. Add workflow tags.
    3. Set the TimeBack benchmark.


 

Alternatively, publish from the Workflows page by accessing the actions menu of the desired workflow, and clicking Publish Workflow.
