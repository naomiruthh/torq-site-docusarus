Tags help you to easily group and categorize workflows. For example, you might group workflows by use case, security framework, SOC team, and many more.

Apply tags to a workflow

Add tags when publishing a workflow, either by typing them into the Tags box, selecting from existing tags in the dropdown, or clicking on any of the auto-generated suggested tags. You can add tags from several locations: 

Publish a workflow

Add suggested tags, type to search for existing tags, or add a new one to the tags list.

Workflows page

In the Tags column, type to search for existing tags, or add a new one to the tags list.

Sort workflows by tags

You can sort  by tags in several places:

Insights dashboard

When you filter by tags, the dashboards present data for workflows with those tag(s).


Workflows page

Find related workflows in the main Workflows page.
