When using Python scripts with workflow data in JSON format, it's important to escape the JSON string properly. Escaping special characters helps prevent code injection attacks and ensures the JSON data is valid.

There are a couple of ways you can escape JSON in Torq.

  • jsonEscape function
  • Escape JSON string step
